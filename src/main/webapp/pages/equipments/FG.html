Function Generator

Instrument Features

  • USB, GPIB and LAN interfaces included in the 33220A function generator
  • Graph mode for visual verification of signal settings
  • Option 001 provides multiple-unit link for synchronous channels
  • The Arbitrary Waveform Generator is LXI Class C Compliant


Waveform Generation

  • 20 MHz sine and square waveforms
  • Ramp, triangle, noise, pulse generation with variable edge, and DC waveforms
  • 14-bit, 50 MSa/s, 64 Kpoint arbitrary waveforms
  • AM, FM, PM, FSK, and PWM Modulation, Linear & logarithmic sweeps and burst all standard

How to use Function Generator


Select type of wave form


Enter frequency value


Select amplitude (Vpp)


Select offset value


After setting all input values click on update display button
