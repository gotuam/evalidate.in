RC Integrator

Procedure

  1. To perform the live experiment on the lab, the student needs to login, or create a login id (if new).
  2. After logging on, the student should select any one option from list of labs
    a) Slot System
    b) One Shot system
    3. If you are selecting SLOT SYSTEM, in that case you have to book your time before performing the experiment; otherwise you can perform experiment directly on ONE SHOT SYSTEM.
  3. To perform the experiment on Slot System, student has to book their slot time. Student should click on Slot Booking
    4. .
  4. As student will click on Slot Booking, a new window will appear which has all slot timing. Student can select only one slot in a single day.
  5. After selection of slot, you should click on Submit button.
  6. As you will click on Submit button, an acknowledgment pop up window will open to give the information that slot has been selected, click on OK button.
  7. Now student has to click on Home button.
  8. Now click on Slot System.
    9. After clicking on Slot system, you are ready to perform the experiment.
  9. Now select the experiment RC Integrator from the experiment list.
  10. After selecting the experiment you will see the schematic and breadboard circuit diagram. The bread board circuit diagram is an example how to connect components on board. Student can connect the components by drag and drop process. Before starting, you should click on Lock button, by doing this your slot time will be locked which means only you have access to the hardware at that time.
  11. Now start to connect the components on Breadboard by drag and drop process.
  12. Now connect the components by wires, there are four different colors of wires available. Red, Green, Blue and Black.
  13. When the process of connecting the components is complete, then student should check that all the components are in proper manner, if the any of the components are not in proper manner then delete the respective components and reconnect the circuit. If you want to delete any wire, click on that wire, the color of wire will change to yellow color; now delete it by clicking on Delete button.
  14. After implementation of components on bread board, student must have connect function generator for providing Input signal, Power Supply for providing power to ICs, these are on left side of bread board (if needed). For seeing the value of DC voltage student must have connect the Digital Multimeter, which is on right side of the Bread Board at bottom mentioned as DMM.
  15. When all the components are connected properly, student should click on FG (Function Generator), After clicking on FG , a new window of function Generator will be appear, The all value like as amplitude, frequency, wave (sine, square, ramp, etc.) and DC offset should be proper and with its unit, after that click on Update Display.
  16. Now click on Run button to see the output.
  17. As youâ€™ll click on run button, a new pop up will open, there have three different option, whatever you want to know from these ,you have to click on that option.
  18. After clicking on Voltage, you will get the value of DC voltage in same pop up.
