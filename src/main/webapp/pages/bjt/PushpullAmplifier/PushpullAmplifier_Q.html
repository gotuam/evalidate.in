Push - Pull Amplifier

Questionnair


Q.1     The Push-pull amplifier is free from?
A)    Noise
B)    Second harmonic distortion
C)    Distortion
D)    None of these
Ans : B

Q.2    The maximum theoretical efficiency of a class A Push-pull amplifier is?
A)    55 %
B)    50 %
C)    78 %
D)    100 %
Ans : B

Q.3    The power amplifiers are called?
A)    Small signal amplifiers
B)    Large signal amplifiers
C)    Magnitude amplifiers
D)    None of these
Ans : B

Q.4    The crossover distortion takes place in?
A)    Class A amplifier
B)    Class B amplifier
C)    Class AB amplifier
D)    None of these
Ans : B

Q.5    The efficiency of class AB and class B amplifiers?
A)    Smaller
B)    Greater
C)    Equal
D)    Not Equal
Ans : C
